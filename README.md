# Real-Time Chat Application

A responsive real-time chat application built using React.js, Node.js, Express.js, and Socket.IO.

## 🚀 Features

- Real-time messaging
- Message history
- Responsive UI
- Auto-scroll to latest message
- Timestamps
- WebSocket communication

## 🛠️ Technologies Used

- React.js
- Node.js
- Express.js
- Socket.IO
- CSS

I developed a responsive real-time chat application using React.js and Socket.IO with WebSocket-based communication, message history handling, auto-scroll functionality, and responsive UI design.
